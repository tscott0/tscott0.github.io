Nested datatypes in Elasticsearch

Posted on October 31, 2018

Create the index

PUT /products

{
    "mappings" : {
        "_doc" : {
            "properties" : {
                "tag" : { "type" : "nested" }
            }
        }
    }
}

Add some products

A tomato is red and round.

PUT /products/_doc/1

{
  "title" : "tomato",
  "tag" : [
    {
      "name" : "red"
    },
    {
      "name" : "round"
    }
  ]
}

An apple is green and round (mostly)

PUT /products/_doc/2

{
  "title" : "apple",
  "tag" : [
    {
      "name" : "green"
    },
    {
      "name" : "round"
    }
  ]
}

Searching for round things

POST /products/_search

{
  "query": {
    "nested": {
      "path": "tag",
      "query": {
        "bool": {
          "must": [
            { "match": { "tag.name": "round" }}
          ]
        }
      }
    }
  }
}

Searching for a green apple

{
  "query": {
    "bool": {
      "must": [
        {
          "match": {
            "title": "apple" 
          }
        },
        {
          "nested": {
            "path": "tag", 
            "query": {
              "bool": {
                "must": [ 
                  {
                    "match": {
                  "tag.name": "green"
                    }
                  }
                ]
              }
            }
          }
        }
      ]
}}}
